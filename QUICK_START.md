# ⚡ Quick Start - Início Rápido

> Quer começar AGORA? Este é o guia para você! 🚀

---

## 🎮 Em 30 Segundos

### 1. Abra o Projeto
```
Duplo clique em: index.html
```

### 2. Busque um Pokémon
```
Digite na barra: pikachu
Pressione: Enter
```

### 3. Aproveite! 🎉
```
- Clique em: < Prev / Next >
- Clique no ícone: 🎵 (música)
- Digite: wellington (easter egg)
```

## ⚠️ Se Algo Não Funciona

1. **Imagens não carregam?**
   → Verifique sua internet

2. **Página em branco?**
   → Pressione F5 (Refresh)

3. **Música não toca?**
   → Clique na página primeiro

4. **Mais problemas?**
   → Veja [TROUBLESHOOTING.md](TROUBLESHOOTING.md)
